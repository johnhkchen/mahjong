## Context

One label dictionary module in `src/app/` mapping every user-facing game term to the two
terminologies — romanized Japanese (current labels, the default) and Traditional Chinese
(吃 碰 槓 胡 自摸 立直 流局 聽牌 振聽, seats 東南西北, score-screen terms) — and every
hardcoded term in the views (ClaimPrompt, Table, hand-end/score screen, prompts) rewired
through it. Core stays byte-for-byte untouched.

_Advances: P2, P4_

## Acceptance Criteria

- [ ] A single dictionary module covers all game terms with both terminologies; no

  default terminology every existing SSR/drive test passes unchanged.
