## Context

The reporter's half: a header "report bug" affordance opening a dialog with a freeform
message field and an auto-attached, read-only report — the serialized notation of the
CURRENT GameRecord plus context (terminology, hand index, action count, app origin).
Two delivery buttons: "copy report" (clipboard, the always-works path) and "open GitHub
issue" (prefilled new-issue URL for johnhkchen/mahjong with title + body); when the
URL would exceed ~6k chars the issue button still opens with a short body instructing
paste-from-clipboard, and the dialog says so. Dictionary terms for all labels.

_Advances: P5, P2_

  clipboard-first body past the length threshold; game input underneath is paused or
  inert while the dialog is open; both terminologies covered; single-file gate green.
