## Context

A small header toggle (next to "new game", same visual register, ≥44px target) switches
the active terminology live; the choice persists as one `localStorage` key and is read at
boot (SSR-safe: absent storage falls back to the default).

_Advances: P2, P4_

## Acceptance Criteria

- [ ] Tapping the toggle relabels all visible game terms without disturbing the running
  hand; the choice survives a reload via exactly one localStorage key; SSR renders (no
  localStorage) fall back to the default terminology without warnings.
