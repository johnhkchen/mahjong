## Context

The public contract's text form (architecture.md §2, deferred since E-002): a compact,
versioned, human-readable serialization of a GameRecord — header line (format version +
game seed), then one line per hand's ordered action log in a terse per-action notation —
and its strict parser. Pure `src/core/` module, zero DOM. Design the action encoding as
part of the replay contract: document it in the module header; the version field exists
so the format can evolve without orphaning old reports.
